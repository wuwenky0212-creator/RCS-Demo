# 税务规则配置PRD——输入输出修订稿

> 适用于替换《税务规则配置需求文档》的第4章和第5章。

## 4、输入描述

### 4.1 债券税费规则查询条件

| 字段 | 控件 | 必填 | 规则 |
|---|---|---:|---|
| 国家/地区 | 单选下拉框 | 否 | 可清空 |
| 关联交易流水号 | 文本输入框 | 否 | 支持模糊查询 |
| 债券代码 | 可搜索单选下拉框 | 否 | 从债券基础数据中选择 |
| 是否生效 | 单选下拉框 | 否 | 已生效、未生效，可清空 |

### 4.2 拆借税费规则查询条件

| 字段 | 控件 | 必填 | 规则 |
|---|---|---:|---|
| 国家/地区 | 单选下拉框 | 否 | 可清空 |
| 交易对手分类 | 多选下拉框 | 否 | 多个选项按“或”查询 |
| 是否生效 | 单选下拉框 | 否 | 已生效、未生效，可清空 |

切换Tab时，系统清空另一产品的专属查询条件，避免条件串用。

### 4.3 债券税费规则维护字段

| 字段 | 必填 | 默认值 | 维护规则 |
|---|---:|---|---|
| 国家/地区 | 是 | 印度尼西亚（ID） | 单选 |
| 产品类别 | 是 | 债券 | 根据当前Tab自动带入，不可修改 |
| 债券代码 | 否 | 空 | 可搜索单选；空值表示不限 |
| 关联交易流水号 | 是 | 空 | 文本输入，不允许为空 |
| Acquisition Price（购入价格） | 是 | 空 | 必须大于0，最多4位小数 |
| 税率（%） | 是 | 0.0000 | 取值范围0—100，最多4位小数 |
| 生效日期 | 是 | 系统日期 | 格式为`YYYY-MM-DD` |
| 失效日期 | 是 | 2099-12-30 | 不得早于生效日期 |
| 结算处理方式 | 是 | 影响结算金额 | 固定值，不可修改 |
| 记账处理方式 | 是 | 记账 | 固定值，不可修改 |
| 是否生效 | 否 | 生效 | 复选框 |

债券规则不维护Portfolio、债券分类和交易对手分类。

### 4.4 拆借税费规则维护字段

| 字段 | 必填 | 默认值 | 维护规则 |
|---|---:|---|---|
| 国家/地区 | 是 | 印度尼西亚（ID） | 单选 |
| 产品类别 | 是 | 拆借 | 根据当前Tab自动带入，不可修改 |
| 交易对手分类 | 否 | 空 | 多选；空值表示不限 |
| 税率（%） | 是 | 0.0000 | 取值范围0—100，最多4位小数 |
| 生效日期 | 是 | 系统日期 | 格式为`YYYY-MM-DD` |
| 失效日期 | 是 | 2099-12-30 | 不得早于生效日期 |
| 结算处理方式 | 是 | 影响结算金额 | 固定值，不可修改 |
| 记账处理方式 | 是 | 不记账 | 固定值，不可修改 |
| 是否生效 | 否 | 生效 | 复选框 |

### 4.5 Excel导入

- 支持`.xlsx`、`.xls`，读取第一个工作表。
- 在当前Tab执行导入，系统自动确定产品类别。
- 债券导入须包含：国家/地区、关联交易流水号、购入价格、税率；债券代码可为空。
- 拆借导入须包含：国家/地区、税率；交易对手分类可为空。
- 生效日期为空时取系统日期；失效日期为空时取`2099-12-30`。
- 结算和记账方式按产品固定，不采纳文件中的其他值。
- 导入前校验必填项、税率、购入价格和有效期；校验失败时提示Excel行号和字段。
- 规则编号、创建人、创建日期由系统生成，不从文件导入。

### 4.6 系统生成字段

| 字段 | 生成规则 |
|---|---|
| 规则编号 | 新增、复制或导入时自动生成，用户不可修改 |
| 创建人 | 取当前登录用户 |
| 创建日期 | 取保存成功日期 |

## 5、输出描述

### 5.1 债券税费规则列表

| 输出项 | 说明 |
|---|---|
| 规则编号 | 规则唯一标识 |
| 是否生效 | 展示当前状态并支持切换 |
| 生效日期、失效日期 | 规则有效期 |
| 国家/地区 | 规则适用地区 |
| 债券代码 | 未配置时显示“—” |
| 关联交易流水号 | 规则关联的交易流水 |
| Acquisition Price | 保留4位小数 |
| 税率（%） | 保留4位小数 |
| 结算处理方式 | 固定显示“影响结算金额” |
| 记账处理方式 | 固定显示“记账” |
| 操作 | 编辑、复制、删除；固定在列表右侧 |

### 5.2 拆借税费规则列表

| 输出项 | 说明 |
|---|---|
| 规则编号 | 规则唯一标识 |
| 是否生效 | 展示当前状态并支持切换 |
| 生效日期、失效日期 | 规则有效期 |
| 国家/地区 | 规则适用地区 |
| 交易对手分类 | 多个分类以逗号分隔；未配置时显示“—” |
| 税率（%） | 保留4位小数 |
| 结算处理方式 | 固定显示“影响结算金额” |
| 记账处理方式 | 固定显示“不记账” |
| 操作 | 编辑、复制、删除；固定在列表右侧 |

### 5.3 导入导出结果

- 导入成功：提示成功导入的规则数量并刷新当前Tab。
- 导入失败：提示失败原因、Excel行号及字段，不显示为成功。
- 导出范围：当前Tab、当前查询条件下的全部规则。
- 导出格式：Excel文件；债券和拆借使用各自的字段结构。
- 导出文件可作为相同Tab的导入模板。

### 5.4 操作反馈

| 场景 | 输出 |
|---|---|
| 查询 | 刷新当前Tab列表 |
| 新增/复制/编辑 | 提示成功，关闭弹窗并刷新列表 |
| 字段校验失败 | 在对应字段显示原因 |
| 规则冲突 | 阻止保存或启用，显示冲突规则及重叠日期 |
| 启用/停用 | 提示最新状态 |
| 删除 | 二次确认；成功后刷新列表 |
| 请求失败 | 保留原数据并提示失败 |

### 5.5 税费现金流输出

| 产品 | 输出要求 |
|---|---|
| 债券 | 输出原交易编号、关联交易流水号、规则编号、Acquisition Price、应税资本利得、税率、税费金额、扣减金额、最终结算金额及记账状态 |
| 拆借 | 输出规则编号、本金、税前利息、税率、利息税费扣减额及最终净结算金额 |
